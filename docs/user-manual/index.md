## Overview

Security Management Center (SMC) High Availability provides redundancy and failover capabilities to ensure continuous management of your NGFW infrastructure. This manual guides you through the complete setup and management process.

## Key Features

- **Automatic Failover**: Seamless transition between primary and secondary SMC servers
- **Data Synchronization**: Real-time replication of configuration and policy data
- **Load Balancing**: Distribute management load across multiple SMC instances
- **Zero Downtime**: Maintain management capabilities during server maintenance

## Prerequisites

Before setting up SMC HA, ensure you have:

- Two or more compatible SMC servers
- Network connectivity between SMC nodes
- Sufficient system resources (CPU, RAM, storage)
- Valid SMC licenses
- Understanding of your network topology
